Narsenta — Static Site (GitHub → Vercel)
===========================================
Files:
- index.html  : main page
- style.css   : metallic dark theme + silver headline with shine
- assets/
    logo-narsenta.png  (replace with your metallic logo PNG with transparent background)
    car-road.jpg       (replace with road car image)
    car-front.jpg      (replace with front view image)
    dealer.jpg         (replace with dealership image)

How to use:
1) Upload this folder to a GitHub repo (e.g., 'narsenta').
2) On Vercel: Add New → Project → Import Git Repository → Select your repo → Deploy.
3) Replace images by uploading new ones to /assets with the SAME names above.
